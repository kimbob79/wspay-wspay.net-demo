수기결제 API

가맹점 서버에서 카드 키인(Keyin) 결제를 직접 요청하는 Server-to-Server API. 결제 요청, 내역 조회를 지원합니다.

POST /pay GET /list JSON

결제통보 (Webhook)

결제 승인/취소 시 가맹점 서버로 실시간 HTTP POST 알림을 수신합니다. 재시도 정책 및 데이터 형식을 안내합니다.

payment.approved payment.cancelled
